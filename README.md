# 실시간 단어 찾기 게임 (word-finder-game)

강사가 주제를 정하면 화면에 뜨는 단어들 중 주제에 맞는 단어만 골라 클릭하는 실시간 수업용 게임입니다. 학생들은 각자 스마트폰이나 PC로 접속해 닉네임만 입력하고 바로 참여합니다.

## 게임 방법

1. 강사가 주제(과일 / 동물 / 나라 / 꽃 / 좋은말)와 단어 변경 간격(3초 / 5초 / 10초)을 고르고 게임을 시작합니다.
2. 학생 화면에 단어 카드 20장(정답 8개 + 오답 12개)이 나타나고, 설정한 간격마다 새 단어판으로 바뀝니다.
3. 주제에 맞는 단어를 클릭하면 +1점, 아닌 단어를 클릭하면 -1점입니다.
4. 제한 시간 60초가 끝나면 상위 3명의 순위가 팝업으로 발표됩니다.

## 강사 기능

- 게임 시작 / 게임 종료 / 초기화
- 참가자 점수판 실시간 확인
- 현재 단어판 미리보기
- 초기화를 누르면 참가자 전원이 퇴장 처리되고 이전 기록이 모두 삭제됩니다.

## 기술 구성

- 단일 `index.html` — HTML/CSS/JS (프레임워크 없음)
- Firebase Realtime Database + 익명 인증 — 실시간 동기화
- Vercel Serverless Function (`api/check-admin.js`) — 강사 비밀번호 확인
  - 환경변수 `ADMIN_PASSWORD` 를 Vercel 프로젝트 설정에 등록해야 합니다.
